# Internationalization (i18n)

## Languages

25 languages supported: en, tr, de, fr, es, it, pt, nl, ar, ru, zh, ja, ko, hi, ur, id, ms, fa, el, bg, ro, pl, uk, az, ka

RTL languages: ar, fa, ur

## Structure

```
packages/i18n/
├── src/
│   ├── languages.ts    # Language config
│   └── index.ts        # getMessages(), t()
└── messages/
    ├── en.json         # English base (source of truth)
    ├── tr.json
    └── ...
```

## Protected Brand Names

Never translated: EnteleWALLET, EnteleKRON, ENK, SOVRA, EnergieMIND, ENM, TVK Group, TVK Labs, EnteleVAULT, EnteleSCAN, TVK ID

## RTL Support

- `dir="rtl"` set on `<html>` for ar, fa, ur
- Wallet addresses, hashes, URLs use `<LtrSpan dir="ltr">` 

## Validation

```bash
pnpm i18n:check
```

Compares all locale files against English:
- Missing keys → fail
- Empty values → fail
- TODO / TRANSLATE / lorem ipsum → fail

## Adding a Language

1. Add to `packages/i18n/src/languages.ts`
2. Create `packages/i18n/messages/{code}.json`
3. Import in `packages/i18n/src/index.ts`
4. Run `pnpm i18n:check`

Only show languages in the switcher once 100% of keys exist.
